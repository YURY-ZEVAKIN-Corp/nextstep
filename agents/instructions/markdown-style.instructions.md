## Formatting Rules

- Use ATX headings (`##`, `###`) with a space after `#`
- H1 (`#`) only at file start — one per file
- No skipped heading levels
- Use `---` for horizontal rules
- Tables must have header separator row (`|---|---|`)
- Code blocks must specify language after opening triple backticks
- Use `**bold**` for emphasis, not `__underscores__`
- Lists: use `-` for unordered, `1.` for ordered
- No trailing whitespace
- Single newline at EOF
